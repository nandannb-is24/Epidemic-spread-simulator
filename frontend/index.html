Algorithm Visualization Lab

Modeling Disease Spread
with Graph Algorithms

This lab visualizes how classical graph algorithms operate on real contact networks, using epidemic spread as a concrete, intuitive backdrop. Every screen foregrounds how the algorithm works, step by step.

The centerpiece is Brandes' betweenness centrality, an O(V·E) algorithm for identifying the highest-traffic hub nodes in a network (the “super-spreaders”). It runs a BFS from every vertex and accumulates dependency scores in a backward pass.
